# LocalDevine Website

This directory contains the website files for GitHub Pages.

## Structure

```
docs/
├── index.html          # Landing page
├── download.html       # Download page
├── css/
│   └── style.css      # Stylesheet
└── README.md          # This file
```

## GitHub Pages Setup

1. Go to repository Settings
2. Navigate to Pages section
3. Source: Deploy from a branch
4. Branch: `main` / `docs` folder
5. Save

The website will be available at:
`https://ohmiler.github.io/localdevine/`

## Local Development

Simply open `index.html` in a web browser to preview locally.

## Notes

- Icon images are referenced from `../public/icon.png`
- Make sure the icon file exists in the `public/` folder
- All links are relative and should work on GitHub Pages
